# Queue
먼저 넣은 데이터가 먼저 반환되도록 설계된 메모리 구조(First In First Out:FIFO)<br>
ex. 너버 우선 탐색(BFS) , 작업 스케줄링, 캐시 구현 등에 유용
- 데이터 입력 : ```Enqueue```
- 데이터 출력 : ```Dequeue```
- 삽입 연산이 이뤄지는 곳 : ```rear(리어)``` - Enqueue가 이뤄짐
- 삭제 연산이 이뤄지는 곳 : ```front``` - Dequeue가 이뤄짐
