## Stack 
나중에 넣은 데이터를 먼저 반환하도록 설계된 메모리 구조(Last In First Out:LIFO)<br>
**데이터를 임시 저장**할 때 사용하는 자료구조<br>
- 데이터 입력 : push
- 데이터 출력 : pop
- 스택에서 입출력이 이뤄지는 부분 : top
- 반대쪽 부분 : bottom

**※ 리스트와 사용** 
- push는 ```append()```,pop은 ```pop()```을 사용
```python
a = [1,2,3,4,5]
a.append(6)
a.append(7)
a.pop()
>>>20
a.pop()
>>>10
```

**※글자역순출력(예시)**
```python
#글자 역순 출력
word = input()
word_list = list(word)
for i in range(len(word_list)):
  print(word_list.pop()) #하나씩 빼면서 출력
```
